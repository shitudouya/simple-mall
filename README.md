

## 简易购物商城

线上地址：[http://mall.tudoublog.com](http://mall.tudoublog.com)

主要技术栈：`React + Redux + Ant Design + Axios + Sass`

后端代码地址：[https://gitee.com/sexylalal/mx_project](https://gitee.com/sexylalal/mx_project)

#### 1.安装

```shell
npm install
```

#### 2.运行

```shell
npm start
```

#### 3.打包

```shell
npm run build
```
